### Creating a component
> 1. Create a function. 
> 2. Add your HTML inside.
> 3. Export it as default. 

Here's the code snippet - 
```
export default function Profile() {
  return (
    <img
      src="https://i.imgur.com/MK3eW3Am.jpg"
      alt="Katherine Johnson"
    />
  )
}
```
